# IMDB-Sentiment-Analysis
This project was completed by Michael Fillinger for CSI 4130 at Oakland University
The goal of this project is to classify movie reviews as positive or negative using machine learning and natural language processing techniques
The dataset used comes from Kaggle and is titled *IMDB Dataset of 50k Movie Reviews*.
Dataset source:https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews?resource=download
## How to Run
1. Make sure the following files are in the same folder:
`train_sentiment_model.py`
`predict_review.py`
`best_sentiment_model.pkl`
`IMDB Dataset.csv`
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
3. Train the model:
   ```bash
   puthon train_sentiment_model.py
4. Predict the sentiment of a review:
   ```bash
   pyton predict_review.py
